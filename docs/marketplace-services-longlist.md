# Locore 駐在員サービス ロングリスト

> 在外邦人（駐在員）が「現地での時間・人脈・言語・文化バイカルチャー」を武器に提供できるサービスを、買い手別・スケール別に列挙。`docs/marketplace-design.md` §13 の二層モデル前提。

---

## 駐在員の構造的優位（出品価値の源泉）

1. **現地での時間・身体性**: その場に居る／行ける／待てる
2. **日本語ネイティブ × 現地語**: 二言語マッチング
3. **文化バイカルチャー**: 「日本人がつまずきがちな所」と「現地のコード」の両方を知っている
4. **現地人脈**: 現地のドアを知る人にしか開かない領域に入れる
5. **現地での生活費 vs 日本側の購買力**: 為替差を価値変換できる
6. **本業スキル**: 駐在中も本業を持っており、それ自体が売れる
7. **失敗の蓄積**: 行政・医療・教育で「自分が踏んだ地雷」がコンテンツ化できる
8. **逆参勤交代の橋**: 帰国後も「日本側に居る現地経験者」として価値が残る

## 開発フェーズへの示唆

### MVP β（最初の 5 カテゴリ）— PRD §15 Week 8 想定

以下 5 つで両看板（買付 + アクセス）+ 流動性エンジン + 専門職を全カバー:

1. **B1: 買付・代理購入**（流動性 + 越境エスクロー検証）
2. **B2: 発送・転送**（買付の補完、ハンドリング訓練）
3. **A1: 案内・同行**（旅行者の入口、ガイド資格 U1 の試金石）
4. **E1: ファッションアクセス**（高単価・収益の柱の証明）
5. **F1: 撮影**（記事 PR が一番効くカテゴリ、F の代表例）

- D1 D2 D3 D4: 同胞向けの低単価層をまとめて投入

- E2 E3 E4: ワイン / ガストロノミー / アート（パリ拡張）

### β + 半年
