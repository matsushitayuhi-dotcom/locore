## 0. ドキュメント概要

本書は Locore のパリ単都市先行ローンチに向けた、M0 から M12 までの実行計画である。月単位ロードマップ、最初の 3 ヶ月分のスプリント計画、採用計画、クリティカルパス、リスクと依存関係を定義する。

時間軸の前提：
- **M0 = 2026-06**（プロジェクト本格始動月）
- **M3 = 2026-09**（クローズドベータ開始）
- **M6 = 2026-12**（パブリックローンチ）
- **M12 = 2027-06**（パリ確立、ロンドン/NY 準備完了）

数値目標は PRD §11.4 に準拠：
- M3：書き手 50 / 記事 100 / MAU 1,000 / GMV ¥100K
- M6：書き手 150 / 記事 300 / MAU 5,000 / GMV ¥1M
- M12：書き手 500 / 記事 1,000 / MAU 30,000 / GMV ¥10M
