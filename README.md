# Locore（ロコレ）

> **Local + Lore** — 現地民が語る、その土地の本当の物語。

在外邦人による旅行情報マーケットプレイス。映えに飽きた中〜上級旅行者と、現地での知見をマネタイズしたい在外邦人を、信頼性ある記事マーケットプレイスで接続する。

## 🔧 確定済みの主要設計判断

- **プラットフォーム**：Web (Next.js + PWA) 単一構成、ネイティブアプリは Phase 3+ 条件付き
- **手数料**：認証クリエイター 10% / 通常 20%（2倍差）
- **居住認証**：Tier S/A/B の3階層
- **評価軸**：ローカル度（0-100）+ 訪問満足度（★1-5）の2軸
- **マップ**：Google Maps Platform Cloud Styling、明るめモダン基調
- **ファウンディング特典**：先着50人、6ヶ月手数料0% + 永久バッジ
- **ローンチ形態**：オープンローンチ（招待制ではない）
- **AI モデレーション**：3次元スコア（観光地依存度・視覚装飾度・文体浅薄度）
- **編集チーム**：内製チーフ + パリ在住フリーランスのハイブリッド
- **クライシス情報**：M0-6 手動キュレーション → M7-12 半自動 → Phase 2 完全自動
