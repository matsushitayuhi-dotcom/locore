# Locore ロゴ・ファビコン用アセット

ここに以下のファイルを配置すると、サイトのヘッダー / SNS シェア /
アプリアイコン / PWA まで一括で反映されます。

すべて **採用済みの「アーチ扉 + 中心ドット」アイコン**と
**"LOCORE" 大文字セリフのワードマーク**を元に作成してください。

## 必要なファイル一覧

| ファイル名 | サイズ | 用途 | 背景 |
|---|---|---|---|
| `logo-wordmark.png` | 横長（例 1200×320） | サイトヘッダー / フッター / ログイン画面の上部 | 透過 |
| `logo-icon.png` | 1024×1024 | フォールバック / 大型表示 | 透過 or クリーム |
| `apple-touch-icon.png` | 180×180 | iOS ホーム画面追加時のアイコン | クリーム（透過不可） |
| `icon-192.png` | 192×192 | Android / PWA インストール時 | クリーム |
| `icon-512.png` | 512×512 | Android / PWA スプラッシュ | クリーム |
| `favicon.ico` | 16×16 + 32×32 マルチサイズ | ブラウザタブ / ブックマーク | 透過可 |
| `og-image.png` | 1200×630 | OGP（Twitter / Facebook / LINE シェア時のサムネ） | クリーム + 中央にアイコン + 下に locore + 短い説明文 |

## 補足

### `og-image.png` の構成案

```
┌────────────────────────────────────────────────────┐
│                                                    │
│                      ▲                              │
│                      🏠                             │
│                                                    │
│                  LOCORE                            │
│        現地に住む人だけが書ける旅行誌                    │
│                                                    │
└────────────────────────────────────────────────────┘
```

Ideogram で生成するときのプロンプトは別途。

### `favicon.ico` の作り方

1. `logo-icon.png` (1024×1024) を https://realfavicongenerator.net にアップロード
2. 生成された favicon.ico をここに置く

### 配置後の動作

ファイルを置いて `pnpm dev` を再起動すると：
- ヘッダー / フッターのテキスト「Locore」が画像に切り替わる（`logo-wordmark.png` 検出時）
- ブラウザタブにアイコンが出る
- スマホで「ホーム画面に追加」したときに `apple-touch-icon.png` が使われる
- Twitter / Facebook シェアで `og-image.png` がカードに表示される

### 画像が無いときの挙動

ファイルが無くてもサイトは壊れない設計です。`<Logo />` コンポーネントは
画像読み込みに失敗したら自動で **terra-cotta 色のセリフ体テキスト** に
フォールバックします。
